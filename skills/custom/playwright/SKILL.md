# Playwright CLI

Use CLI-driven browser automation for interactive verification tasks.

## Workflow

1. Verify runtime prerequisites.
2. Open target URL in headed mode when visual confirmation matters.
3. Capture snapshots/screenshots and extract structured results.
4. Report reproducible command sequence used.

## Scope

- Navigation and interaction flows
- Form entry and verification
- Visual snapshots and screenshots
- Data extraction from rendered pages
